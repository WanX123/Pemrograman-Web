# Daftar Riwayat Hidup

Website CV pribadi sederhana dan modern untuk menampilkan data diri, kontak, riwayat pendidikan, media sosial, dan daftar keahlian.

## Preview

Project ini berupa halaman web statis dengan tema dark mode, layout responsif, sidebar profil, kartu informasi, timeline pendidikan, dan badge skill interaktif.

## Fitur

- Tampilan CV modern dengan dark theme
- Layout responsif untuk desktop dan mobile
- Foto profil dan informasi kontak
- Link media sosial Instagram dan GitHub
- Timeline riwayat pendidikan
- Badge keahlian dengan efek hover
- Animasi sederhana saat halaman dimuat

## Teknologi

- HTML5
- CSS3
- JavaScript
- Google Fonts: Inter
- Font Awesome Icons
